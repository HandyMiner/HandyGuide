HandyMiner King Cobra - 快速入门指南
  • 下载 HandyMiner Release ZIP 或下载 HandyMiner Release Tarball
  • 先决条件：
    • 安装 nodeJS
    • （仅限 windows）将 git bash 安装到 Program Files/Git
  • 运行：
    • Windows：双击“dashboard.windows.bat”
    • Linux：双击“dashboard.sh”
    • Mac：双击“dashboard.mac.command”
    • CLI： 或
  • 配置：首次运行将启动配置器
    • 选择 GPU
    • 设置模式（单独 | 库）
    • 强度：11（对于野兽模式）
    • 库难度：-1（对于可变难度）
    • 库主机：
      • f2pool 端口：
        用户名：wallet.workerName
        密码：anything
      • 6block 端口：
        用户名：registeredMinerName
        密码：anything
      • PoolFlare 端口：
        用户名：wallet.workerName
        密码：anything
      • hnspool 端口：
        用户名：registeredUsername
        密码：registeredUserPassword
  • 钱夹：
    • Bob Wallet（GUI，非监禁）
