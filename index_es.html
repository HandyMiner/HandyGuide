HandyMiner King Cobra - Guía de inicio rápido
  • Descargar HandyMiner Versión ZIP o Descargar HandyMiner Versión Tarball
  • Requisitos previos:
    • Instalar nodeJS
    • (solo para Windows) Instalar git bash en Archivos de programa/Git
  • Ejecución:
    • Windows: hacer doble clic en 'dashboard.windows.bat'
    • Linux: hacer doble clic en 'dashboard.sh'
    • Mac: hacer doble clic en 'dashboard.mac.command'
    • CLI: o
  • Configuración: La primera ejecución iniciará el configurador
    • Seleccionar los GPU
    • Elegir un modo (individual | grupal)
    • Intensidad: 11 para un modo óptimo
    • Dificultad del grupo: -1 para una dificultad variable
    • Anfitriones del grupo:
      • f2pool puerto:
        nombre de usuario: billetera.nombreTrabajador
        contraseña: cualquiera
      • 6block puerto:
        nombre de usuario: nombreMineroRegistrado
        contraseña: cualquiera
      • PoolFlare puerto:
        nombre de usuario: billetera.nombreTrabajador
        contraseña: cualquiera
      • hnspool puerto:
        nombre de usuario: nombreUsuarioRegistrado
        contraseña: contraseñaUsuarioRegistrado
  • Billeteras electrónicas:
    • Bob Wallet (Interfaz Gráfica de Usuario, sin custodia)
