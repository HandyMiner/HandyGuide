HandyMiner King Cobra - Schnellstart
  • Downloade HandyMiner als ZIP oder Downloade HandyMiner als Tarball
  • Vorbereitung:
    • Installiere nodeJS
    • (Nur Windows) Installiere git bash in Programme/Git
  • Starten:
    • Windows: Doppelklick auf 'dashboard.windows.bat'
    • Linux: Doppelklick auf 'dashboard.sh'
    • Mac: Doppelklick auf 'dashboard.mac.command'
    • Kommandozeile: oder
  • Konfiguration: Beim ersten Start wird der Konfigurator initialisiert
    • Wähle GPUs
    • Wähle einen Modus ( solo | pool )
    • Intensität: 11 für Biestmodus
    • Pool Schwierigkeit: -1 für variable Schwierigkeit
    • Pool Hosts:
      • f2pool Port:
        Nutzername: wallet.workerName
        Passwort: anything
      • 6block Port:
        Nutzername: registeredMinerName
        Passwort: anything
      • PoolFlare Port:
        Nutzername: wallet.workerName
        Passwort: anything
      • hnspool Port:
        Nutzername: registeredUsername
        Passwort: registeredUserPassword
  • Wallets:
    • Bob Wallet (GUI, Non-Custodial)
